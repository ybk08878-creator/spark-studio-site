需要上傳的檔案：
- index.html
- styles.css
- script.js
- assets/software-dashboard.svg

GitHub Pages 發布方式：
1. 到 https://github.com/ 登入你的帳號。
2. 建立一個新的 repository，例如 spark-studio-site。
3. 進入 repository 後，按 Add file，再按 Upload files。
4. 把這個資料夾裡的 index.html、styles.css、script.js、assets 資料夾全部拖進去。
5. 按 Commit changes。
6. 到 Settings，點 Pages。
7. Source 選 Deploy from a branch。
8. Branch 選 main，資料夾選 /root，按 Save。
9. 等 1 到 3 分鐘，GitHub 會顯示你的網站網址。
